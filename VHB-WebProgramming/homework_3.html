Aufgabe 1 – PHP – Serverseitige Validierung

Erweitern Sie das bereits von Ihnen bereits erstellte HTML5-Formular um eine serverseitige Validierung.
Zur Erinnerung: Es wurden die folgenden Daten erfasst:

  • Vorname (*)
  • Nachname (*)
  • Matrikelnummer (genau 7 Ziffern) (*)
  • Telefonnummer (aus Deutschland) (*)
  • E-Mail-Adresse (*)
  • Geburtsdatum (Teilnehmer muss mind. 18 Jahre alt sein) (*)
  • Angabe des Studiengangs („Wirtschaftsinformatik“, „Wirtschaftswissenschaften“): Es muss genau ein Studiengang ausgewählt werden. (*)
  • Auswahl von Klausuren („Web-Programming“, „E-Wiinf“, „Logistik 1“): Es muss mindestens eine Klausur ausgewählt werden (*)
  • Erfassungsfeld für weitere Anmerkungen

Stellen Sie nun auch serverseitig sicher, dass durch das Formular nur gültige Inhalte erfasst werden können und auch alle Pflichtfelder (*) ausgefüllt werden.
Bei fehlenden oder falschen Eingaben sollen entsprechende Fehlerhinweise erscheinen.
Dass Benutzer bei fehlerhafter Eingabe nicht alle Daten erneut eingeben müssen, sollen die bereits abgesendeten Daten im Attribut „value“ beim entsprechenden Eingabefeld voreingetragen werden.
TIPP: Schalten Sie zur Überprüfung Ihres Codes die HTML5-Formular-Validierung mithilfe von novalidate=“novalidate“ aus.
