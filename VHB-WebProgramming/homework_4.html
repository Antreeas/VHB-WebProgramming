Aufgabe 1 – PHP – Body Mass Index (BMI)

Erstellen Sie ein Formular, das zwei Eingabefelder (zur Erfassung von Körpergröße und Körpergewicht) umfasst.
Formatieren Sie das Formular mit Ihnen bekanntem HTML/CSS-Code. Validieren Sie die Eingaben sowohl client- als auch serverseitig.
Erstellen Sie eine Funktion (bmi_calc), die auf Grundlage der eingegebenen Werte den BMI errechnet und ausgibt [BMI = Körpergewicht / (Körpergröße in m)²].
Runden Sie den errechneten BMI auf eine Nachkommastelle.
Formatieren Sie die Text-Ausgabe „Ihr BMI beträgt X“ gemäß folgendem Schema:

Errechneter BMI Farbe
≤ 18.5 blue
18.6-25.0 green
25.1-30.0 yellow
> 30.0 red

Erstellen Sie dazu ebenfalls eine Funktion (bmi_color).
